## modelrelay (Original)

This project is a Python port of **modelrelay**, originally created by Elliptic Marketing.

- Original repository: https://github.com/ellipticmarketing/modelrelay
- License: MIT (same as this port)
- Core concepts borrowed:
  - Model scoring system with benchmark-based quality ratings
  - Provider abstraction layer with health checking
  - Auto-routing with fallback chains
  - OpenAI-compatible API design
  - CLI and server architectures

All code in this repository has been rewritten in Python from scratch, but the design, API surface, and naming conventions intentionally mirror the original to ensure compatibility and respect the original work.
